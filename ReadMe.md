```
pip install virtualenv
```

```
.venv\Scripts\activate
```

```
pip install -r requirements.txt
```


```
pip install streamlit
```

##### streamlit
```
pip install streamlit
```

```
pip install pipreqs
pipreqs . --force --encoding=latin1 --ignore=.venv
```
